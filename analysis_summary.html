BIOLOGICAL ANALYSIS DASHBOARD - DESIGN REFRESH

PROBLEM TO SOLVE

This is a small part of a complex bioinformatics application that displays large amounts of data in the user interface.

In an effort to reduce the waiting time for users, the whole process of loading a biological analysis was rethought. This involved reconstructing of the analysis summary page to fit the new data acquisition process.

Even though there were no specific requirements to change the look and feel of the new implementation, I saw this as an opportunity and a challenge to improve it by doing small changes that wouldn't affect the users already familiar with the dashboard.

THE SOLUTION

The main goal was to improve the readability of the page so that the users would get a quicker and better understanding of the information presented to them.

I broke this down into a list of small improvements:

  • A better separation of the sections by using solid color headers instead of the fading gradient and a drop shadow effect
  • Less intrusive and better placed expand/collapse icons
  • Added some indent to the content of each section to enforce the information hierarchy
  • Dimmed the black horizontal separators so that they donâ€™t break the visual hierarchy
  • improved export buttons
  • improved name and display of overlap column
  • improved display of the p-values and p-value ranges

CONCLUSION

I consider this little project a success because I was able to improve the usability of the dashboard with minor changes to the look and feel. The design was also relatively easy to implement.

I got mostly positive feedback from the people seeing the improved dashboard for the first time and some of the beta testers were kind enough to provide written feedback:

"This is a really nice page that can make more sense of the results. Like the distribution and the more detailed info for the ratio/overlap."

"Everything looks great on my mac OSX10.9.3. I launched from the Beta installer. I love the sleeker look of the New Summary."

The most challenging part of the project was the p-value chart and it is covered in depth in another article since it is a novel visualization and it's conception is an interesting story itself.
